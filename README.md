# SmartSeason

Full-stack farm monitoring system

## Tech Stack
- Node.js (Express)
- MySQL (Railway)
- React (Frontend)
- Railway (Backend)
- Render (Frontend)

## Setup
### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start

## Features
- User authentication
- Field tracking
- Field updates
- Role-based access (admin/agent)
